## 運用メモ

- リポジトリ: `nonimon3/onsa` (public)
- 本番URL: https://nonimon3.github.io/onsa/
- 更新: `npm run deploy` で dist/ を再ビルド→gh-pages push
- QR: `npm run qr:pages` で本番URLのQRを表示
- ローカル開発: `npm start` （同一Wi-Fi PCでの dev サーバー + LAN QR）
- パック単体: `npm run pack` で `onsa.ehpk` を生成

## 追加メモ
- Vite HTTPS: `npm run dev:https` で自己署名HTTPSサーバー起動
- QR: `npm run qr:https` で HTTPS 向けQR生成
- `even-dev` への登録は必須ではない（実機テストには不要）— 別途 G2 の動作をシミュレーターで事前確認したい場合のみ

### 判断メモ
- ピッチ検出は YIN ではなく自己相関+放物線補間を採用（実装コスト/精度のバランス）
- UI 言語は日本語ベース（音名は国際表記 A/B/C…）
- `@evenrealities/even_hub_sdk` (公式SDK) を直接使用（timerと同じ方針）で十分
  - base_app の EvenBetterSdk ほどの抽象化は必要ない
  - ただし AppModule として even-dev に載せるため index.ts / main.ts は共通パターンに従う
